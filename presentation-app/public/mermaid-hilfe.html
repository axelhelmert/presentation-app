🎨 Mermaid Diagramm Hilfe

📝 Grundlagen

Mermaid-Diagramme werden in Markdown mit folgendem Format eingefügt:

```mermaid graph LR A --> B ```
Tipp: LR = Left to Right (von links nach rechts), TD = Top Down (von oben nach unten)

📦 Formen

Rechteck

```mermaid graph LR A[Rechteck] ```
graph LR A[Rechteck]

Abgerundetes Rechteck

```mermaid graph LR A(Abgerundet) ```
graph LR A(Abgerundet)

Kreis

```mermaid graph LR A((Kreis)) ```
graph LR A((Kreis))

Raute (Entscheidung)

```mermaid graph LR A{Entscheidung} ```
graph LR A{Entscheidung}

Parallelogramm

```mermaid graph LR A[/Input/] B[\Output\] ```
graph LR A[/Input/] B[\Output\]

Sechseck

```mermaid graph LR A{{Sechseck}} ```
graph LR A{{Sechseck}}

➡️ Pfeile und Verbindungen

Einfacher Pfeil

```mermaid graph LR A --> B ```
graph LR A --> B

Pfeil mit Text

```mermaid graph LR A -->|Text| B ```
graph LR A -->|Text| B

Gestrichelte Linie

```mermaid graph LR A -.-> B C -.Text.-> D ```
graph LR A -.-> B C -.Text.-> D

Dicke Linie

```mermaid graph LR A ==> B C ==Text==> D ```
graph LR A ==> B C ==Text==> D

Linie ohne Pfeil

```mermaid graph LR A --- B C ---|Text| D ```
graph LR A --- B C ---|Text| D

Bidirektionale Pfeile

```mermaid graph LR A <--> B C <-.-> D ```
graph LR A <--> B C <-.-> D

🎨 Farben

Elemente können mit CSS-Styling eingefärbt werden:

Einzelne Elemente einfärben

```mermaid graph LR A[Start] B[Prozess] C[Ende] A --> B --> C style A fill:#10b981,stroke:#059669,stroke-width:2px,color:#fff style B fill:#3b82f6,stroke:#2563eb,stroke-width:2px,color:#fff style C fill:#ef4444,stroke:#dc2626,stroke-width:2px,color:#fff ```

Häufig verwendete Farbcodes:

Farbe Hex-Code (Fill) Hex-Code (Stroke/Dunkel) Verwendung
Grün #10b981 #059669 Start, Erfolg, Positiv
Blau #3b82f6 #2563eb Prozesse, Aktionen

Style-Optionen:
• fill - Hintergrundfarbe
• stroke - Rahmenfarbe
• stroke-width - Rahmendicke in Pixeln
• color - Textfarbe
• width - Breite in Pixeln
• height - Höhe in Pixeln

📏 Größe steuern

Mit width und height kannst du die Größe der Formen festlegen. Das ist besonders nützlich, um alle Elemente gleich groß zu machen.

Alle Rechtecke gleich groß

```mermaid graph LR A[Kurz] --> B[Mittellanger Text] --> C[X] style A width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff style B width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff style C width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff ```
graph LR A[Kurz] --> B[Mittellanger Text] --> C[X] style A width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff style B width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff style C width:200px,height:60px,fill:#3b82f6,stroke:#2563eb,color:#fff

Klassen für einheitliche Größen

```mermaid graph TD A[Element 1] B[Element 2] C[Element 3] D[Wichtig!] A --> B --> C A --> D --> C classDef standard width:180px,height:50px,fill:#3b82f6,stroke:#2563eb,color:#fff classDef wichtig width:180px,height:50px,fill:#ef4444,stroke:#dc2626,color:#fff class A,B,C standard class D wichtig ```
graph TD A[Element 1] B[Element 2] C[Element 3] D[Wichtig!] A --> B --> C A --> D --> C classDef standard width:180px,height:50px,fill:#3b82f6,stroke:#2563eb,color:#fff classDef wichtig width:180px,height:50px,fill:#ef4444,stroke:#dc2626,color:#fff class A,B,C standard class D wichtig
Tipp: Mit classDef definierst du wiederverwendbare Klassen. Diese kannst du dann mit class A,B,C klassenname auf mehrere Elemente anwenden. So bleiben alle Formen konsistent!

💡 Tipps & Tricks

  • Mehrzeilige Labels: Verwende <br> für Zeilenumbrüche in Labels
  • Sonderzeichen: Verwende Anführungszeichen bei Sonderzeichen: A["Text mit (Klammern)"]
  • Ausrichtung: LR (Links-Rechts), RL (Rechts-Links), TD (Top-Down), BT (Bottom-Top)
  • Mehrere Verbindungen: A --> B & C & D verbindet A mit B, C und D
  • Subgraphen: Gruppiere Elemente mit subgraph für visuelle Bereiche
  • Einheitliche Größen: Nutze classDef und class um mehrere Elemente mit denselben Eigenschaften zu versehen
